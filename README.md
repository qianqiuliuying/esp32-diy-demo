# esp32-diy-demo

esp32嵌入式小项目

\## ex01项目

功能1：基础HelloWorld输出程序

功能2：新增scanf键盘输入，回显用户输入数据

\## lab01项目

芯片上的小灯闪烁并输出hello，esp32

\## lab02项目

版本一：芯片上的小灯闪烁，亮时输出LED ON，灭时输出LED OFF

版本二：芯片上的小灯呈现”三短三长三短“的闪烁，实现SOS信号

\## lab03项目

版本一：呼吸灯小实验

版本二：三个呼吸灯交替小实验

\## ex02项目

使用millis()函数实现1Hz频率小灯闪烁

\## ex03项目

使用millis()函数实现SOS”三短三长三短“信号
